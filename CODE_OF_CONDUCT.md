# Code of Conduct

This project follows the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be civil. Assume good faith. Disagree on the work, not the person. Harassment, personal attacks, and demeaning language are not tolerated.

## Scope

Applies in all project spaces: the GitHub repo (issues, PRs, discussions, code), any chat channel run on behalf of the project, and any public event where someone is representing the project.

## Reporting

If something needs to be raised privately, email the maintainer listed in [SECURITY.md](SECURITY.md). Reports are handled discreetly.

## Enforcement

Maintainers may warn, hide, lock, or ban contributors who violate this Code of Conduct. Decisions are documented and proportionate.
